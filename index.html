      坐标广州，工作经验一年，跑了5天的面试，一天最多两家，拿了三家offer，薪资还可以，很多公司我第一轮就没过，还好有的面试官对我这初级的要求也不是太高，放我一手。有点经验，做个总结。

      # 招聘渠道

      我用过的不多，主要就三个app

      • 拉钩：程序员专用求职app，一键速投，手指稍微点一点就能投数十家，可能是每家公司收到的简历太多了，如果没什么亮点，像我这种初级来说，面试率低的惊人。
      • 猎聘：这个是面向中高端的，各位老哥工作两三年有能力后，可以来试试这个，反正我投是基本没人理我。
      • boss直聘：我主要用的一个软件，初中高级的都有，而且回复率是我接触的最高的一个软件，像我一样是初级的话，可以试试这个。

      # 准备的材料

      十份简历随身带，别怕花钱，身份证（有的公司需要刷身份证访问），一根笔，最重要的，我建议提前准备好体检报告

      # 开场自我介绍

      一些老哥可能全去怼技术问题了，自我介绍这其实比较重要的，差不多就几个点

      # 缓解紧张，让自己进入状态

      我第一次自我介绍连话都没说利索，说话磕磕巴巴的，面试的小姐姐都逗乐了，整个面试下来也没什么技术问题，但自我表达的能力差的不行，这次面试就是在白给。

      # 给面试管点时间了解下你

      说实话基本没人会提前看你简历，自我介绍这段时间也是准备接下来问你啥，所以自我介绍的时候，最好别把自己往坑里引。

      # 自我介绍应该说点什么

      姓名 + 工作时间 + 项目内容 + 工作内容 + 工作技能

      基本就是这么一个套路，搞那些花里胡巧的，面试管估计也嫌烦，如果工作经验不太多，那就从基础下手，都坐这了，面试管又不会把人轰出去，想办法把内容往基础上引，之后就考验各位老哥的本事了

      # 面试管提问环节

      基本两大问题，各位老哥可以准备准备

      # 项目经历

      说实话要是太注重项目经验我基本就凉了，上家公司我基本都是在搞无聊又恶心的业务，对前端技术要求还真不高，最常问的，就是你上个项目有什么难点，怎么解决的。说实话，上个公司的前端问题我基本都能自己解决，一个前端一年能解决的问题，我觉得都不是什么难问题。。。

      # 基础考察

      这个环节对于初级和校招生来说，应该是最重要的一个环节，而且，没有太高的门槛！！！，还可以准备的很充分 至于该怎么准备，easy的不行，有个小程序面经大全，各个方向的基本都有，去上面搞几家大厂的面试题，哪里不会补哪里，没有项目一定要多做demo模拟场景，以防面试管换个花样就不会了，或者面试管只问了你其中一个点，可以把相关有深度的东西都说出来，送面试管个豪华大礼包，有目的有方向的准备几个月，稳的很。

      还有，我建议各位老哥不要裸辞，今年秋招以过大半，再说现在寒冬，倒不如先准备几个月，模拟模拟面试，等明年春招再说，如果像我一样裸辞心态又比较急躁的，可以学我没事找妹子聊聊天。

      # 你有什么要问我的

      这我用的一级标题，还加黑了，这个环节我感觉是最重要的，其实找工作，适合自己最重要，我在收到第一家offer的时候到了这个环节才开始考虑这个问题，结果这个环节我就随便问了问来公司之后需要做的内容就完了，事后我也一直在想这事，总结了下，基本还需要这几点

      如果技术面之后还有人事面，简单的问下自己入职短期内的工作内容，先有个初步的了解，这一步我感觉没必要问太多，人事面还没过呢，想的太早了。 人事面如果差不多可以问下福利待遇，主要是调薪，补贴，加班费，额这我还真不太清楚，各位老哥可以引导HR多说点。 人事面过了之后，基本就妥了，可以再返回去跟技术负责人细聊一下，问一下工作的技术栈看看是否和自己的匹配，说出自己对工作内容的期望，这个最好说一下，看看技术负责人觉得合不合适，万一进了公司发现自己不喜欢，那对自己对公司都很难受，公司未来的发展计划，扩张还是保持现状，人员的规模等等，如果公司同意，或者时间比较充分，我建议各位老哥可以在公司坐一会，观察一下公司的工作情况，还有具体的工作内容

      # 一些乱七八糟的

      这两篇文章都很不错，200W字那篇还有面试指导，而且我收藏夹里有很多收集的干货，有兴趣的可以看看，一天最好一家面试，我开始一天跑两家，过了几天身体就比较难受，而且每天复盘整理经验也要花很多时间，找工作还是急不得，再焦躁也要把自己心态稳下来，体检报告要提前准备好！！！

      # 随便聊聊

      想说的很多，各位老哥们就听我逼逼会，这边其实跟面试没什么关系，算是个工作经历。

      入职刚开始两个月什么活都没有，闲的我害怕，跟我大学导师聊了聊说了个Vue这么个关键词，那时候我连html都不会，vue我都不知道是干什么用的，直接裸上的vue框架，现在想想真是憨，前半年的时间，基本都在打杂写SQL，有时候实在没什么事情，偷偷的跑到厕所楼梯自己在那看前端的视频和书，后来被我经理发现了，叫到工位聊了一个多小时，算是默许了我没事可以看看这些东西，挺感谢我师傅和经理的，之后公司比较缺这方面的，我经理给我推荐了过去，算是正式的开始了我的前端生涯。 过去之后，算是开启地狱模式吧，无休止的996，做事根本没有规范，想怎么写就怎么写，也没有技术把关，东拼西凑的队伍，只需要简单的写写js，剩下的，就是跟智障一样的不确定性业务问题。

      辞职之后我立马面过一两家，但这半年，哪怕是我挤出来时间学的东西，根本用不上，除了原型链和this以及闭包之外，其他的基本没怎么问过，而且面试问的很多东西我也不会，所以说有方向的准备非常重要！！！那段时间算是处于一个心态崩溃的时间，家里人也发现我状态不对劲，就回家休养了一个月，虽然错过了九月这个黄金期，但也静下心来好好准备了一个月，国庆后也算比较顺利。
